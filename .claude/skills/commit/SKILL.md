---
name: commit
description: Writes and pushes a commit containing all of the changes since the last commit. 
allowed-tools:Bash(git commit -m ' *), Bash(git add:*), Bash(git push *)
---

1. Look at commit.md and the git diff and use these sources to compose a commit message for all changes that have been made (you can also use any knowledge that you have of the situation from your current context). This commit message should be comprehensive for all changes that have been made to the repo. 

2. Check with the user to confirm that the commit message looks good. 

3. If the user agrees to the commit, clear commit.md and make the commit using what you wrote.

4. Push the commit to remote
